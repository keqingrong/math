# @keqingrong/math (WIP)

> Utilities for some frequently used calculation

## Installation

```sh
npm install @keqingrong/math
```

## API

- `mb2gb()`
- `cent2yuan()`
- `addYuan()`
- `subYuan()`
- `calculateDiscountedPrice()`
- `calculateDiscount()`
- `toFixed()`
- `toFixedAdvanced()`
- `stripTrailingZeros()`
- `getDecimalDigits()`
- `subDecimal()`
