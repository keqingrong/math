# @keqingrong/math (WIP)

> Utilities for some frequently used calculation

[![npm version](https://img.shields.io/npm/v/@keqingrong/math.svg)](https://www.npmjs.com/package/@keqingrong/math)

## Installation

```sh
npm install @keqingrong/math
```

## API

- `mb2gb()`
- `cent2yuan()`
- `addYuan()`
- `subYuan()`
- `calculateDiscountedPrice()`
- `calculateDiscount()`
- `toFixed()`
- `toFixedAdvanced()`
- `stripTrailingZeros()`
- `getDecimalDigits()`
- `subDecimal()`
